В файле `DbAndQuery` лежат запросы на создание таблиц и запрос на получение статистики.

Заполнение таблицы данными происходит при запуске консольного приложения после нажатия клавиши Enter (при нажатии другой заполнения не произойдет). 
Готовый файл с данными называется `measurements.csv` и будет лежать в бинарниках.
Модели на языке c# были созданы с помощью файла GeopyDb.tt и разбиты по разным файлам.
